The application is built with these JavaScript libraries and tools:
  • React - a library for building user interfaces
  • Redux - a state management library
  • Reselect - a library for caching values of functions of Redux state
  • Babel - a JS transpiler: converts newer and nonstandard features of JavaScript to format compatible with older browsers, additionally converts React markup
  • Webpack - a JS bundler: converts multiple JavaScript files into a single script file, may additionally package other assets
The app is simple enough to not require CSS tools like Sass. All styles are in a single CSS file.
